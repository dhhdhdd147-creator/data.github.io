## Курсовая: Телефоны и абоненты (Flask + MS Access)

Это учебный мини‑проект с CRUD для двух сущностей:

- `Subscribers` (абоненты): ФИО, паспорт/ID, адрес
- `Phones` (телефоны): номер, оператор, статус (активен/заблокирован), связь с абонентом

### Требования

- Python 3.x
- Установленный драйвер Microsoft Access ODBC (например, `Microsoft Access Driver (*.mdb, *.accdb)`)
- База `.accdb` (в файле), к которой приложение подключается через ODBC

### Установка

1. Установите зависимости:
   - `pip install -r requirements.txt`
2. Укажите строку подключения к Access:
   - в переменной окружения `ACCESS_CONNECTION_STRING`
   - пример (замените путь):
     - `DRIVER={Microsoft Access Driver (*.mdb, *.accdb)};DBQ=C:\path\to\database.accdb;Uid=Admin;Pwd=;`

### Создание таблиц

В файле `schema.sql` лежит схема для MS Access.

Варианты:

1. Выполнить `schema.sql` в MS Access (через запросы).
2. Или просто запустить `app.py` — код в `db.py` попробует создать таблицы, если их нет (но в некоторых конфигурациях Access FK/constraint могут не создаваться автоматически).

### Запуск

- `python app.py`
- открыть: `http://127.0.0.1:5000`

### Автосоздание .accdb

Чтобы “создать базу саму по себе” (файл `.accdb` в папке проекта):

1. Убедитесь, что установлены `pyodbc` (и доступен драйвер MS Access ODBC).
2. Установите `pywin32` (нужно для автоматического создания `.accdb`):
   - `pip install pywin32`
3. Выполните:
   - `python create_db.py`

После этого приложение сможет подключиться к `database.accdb` в этой же папке.

### Страницы

- `/` — главная
- `/subscribers` — список абонентов + поиск
- `/subscribers/new` — добавить абонента
- `/phones` — список телефонов + поиск
- `/phones/new` — добавить телефон

### Важные замечания

- При удалении абонента связь у телефонов обнуляется (`Phones.SubscriberId = NULL`), чтобы телефоны не “потерялись”.
- CSRF-защита в этом учебном варианте не добавлялась.
